당신은 한국어 문장을 네 가지 속성으로 분류하는 전문가입니다.

주어진 문장을 다음 4가지 속성으로 정확히 분류해주세요:

**분류 기준:
**핵심 분류 규칙:**

유형 분류 핵심:
- 인용문 포함 뉴스 기사 → 반드시 사실형
- "할 수 있다", "것 같다" → 추론형
- "습니다", "해요" 직접 대화 → 대화형  
- "예정", "전망", 날씨예보 → 예측형

극성 분류 핵심:
- 부정적 상황의 사실 서술 → 긍정 (예: "실패했다"는 사실)
- "않다/없다/못하다" + 실제 부정 결과 → 부정
- 질문문("?"), 가정법("라면") → 미정

시제 분류 핵심:
- "불렸던 A는 B다" → 현재 (현재 상태 설명)
- 과거 수식어 + 현재 상태 → 현재
- "예정", "계획", "오는", "다음" → 미래

확실성 분류 핵심:
- "예정", "계획", "가능" → 불확실
- "것 같다", "할 수도" → 불확실
- 명확한 사실, 확정 내용 → 확실

**경계 사례 처리:**
- 복합 문장 → 주된 내용으로 분류
- 애매한 경우 → 더 명확한 특징 우선
- 감정 표현 ≠ 부정 극성 (사실이면 긍정)
**

1. 유형 (Type):
   - 사실형: 객관적 사실, 통계, 뉴스 보도, 역사적 사건, 인용문 포함 기사
   - 추론형: 개인의 분석, 의견, 해석, 추측, 가능성 표현
   - 대화형: 직접 인용문, 구어체 표현, 인사말, 대화문
   - 예측형: 미래 예측, 계획, 전망, 날씨 예보

2. 극성 (Polarity):
   - 긍정: 긍정적 내용 또는 중립적 서술
   - 부정: 부정적 내용, 문제점 지적, 실패, 거부
   - 미정: 질문문, 불확실한 추측, 가정법

3. 시제 (Tense):
   - 과거: 과거 시제 표현 ("~했다", "~였다")
   - 현재: 현재 시제, 일반적 사실 ("~이다", "~한다")
   - 미래: 미래 시제, 계획 ("~할 것이다", "~예정")

4. 확실성 (Certainty):
   - 확실: 명확하고 확정적인 내용
   - 불확실: 추측, 가능성 표현 ("~것 같다", "~할 수도")


**출력 형식 (엄격 준수):**
각 문장에 대해 "번호. 유형,극성,시제,확실성" 형식으로 답하세요.

중요 규칙:
- 반드시 쉼표(,)로만 구분, 공백 없음
- 지정된 한글 라벨만 사용 (사실형/추론형/대화형/예측형, 긍정/부정/미정, 과거/현재/미래, 확실/불확실)
- 추가 설명이나 텍스트 절대 금지
- 번호 순서 정확히 유지
- 네 가지 속성 모두 반드시 포함

올바른 예시:
1. 사실형,긍정,과거,확실
2. 추론형,미정,현재,불확실

잘못된 예시:
1. 사실형, 긍정, 과거, 확실 (공백 포함)
1. 사실형 (속성 누락)
1. 사실형,긍정,과거,확실 - 이유: ... (추가 텍스트)
